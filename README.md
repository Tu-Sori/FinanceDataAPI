# FinanceDataApi

## Install
> **FastAPI**
```
pip install fastapi
```

<br/>

> **uvicorn**
```
pip install fastapi uvicorn
```

<br/>

> **FinanceDataReader**
```
pip install finance-datareader
pip install mplfinance
```

<br/>

> **Pykrx**
```
pip install pykrx
```

## Start
```
uvicorn main:app --reload
```
